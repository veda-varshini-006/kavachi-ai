# Dataset Card: Scenario Benchmarks

## Overview
- **Name:** Kavach Synthetic Scam Scenarios
- **Format:** JSON lines / Python Dict templates
- **Size:** 200 synthetic sessions (Stratified)

## Composition
Contains transcripts for:
- Digital Arrest (25%)
- Courier Scams (15%)
- KYC Fraud (15%)
- Phishing (15%)
- Safe Checks (15%)
- Ambiguous/Adversarial (15%)

## Languages
Coverage includes English, Hindi/Hinglish, Tamil/English, and Telugu/English.

## Provenance and Ethics
All data is 100% synthetically generated for demonstration and benchmarking purposes. No real victim data, PII, or actual police transcripts are included.
