# Release Notes v1.0.0-rc1

## Major Features
- **Scam Coercion Engine:** Live call transcript processing for threat detection.
- **Counterfeit Screening:** Classical computer vision pipeline for uploaded suspect notes.
- **Graph & Geospatial Intelligence:** Fully offline-capable visualization of fraud rings and privacy-preserving K-anonymized heatmaps.
- **SOC Case Management:** Analyst dashboard with Merkle-chain audit logs and redaction workflows.

## Deprecations
- Removed pure SVG Map dependencies in favor of high-performance `maplibre-gl` and `deck.gl`.

## Known Limitations
- Purely synthetic prototype. "Block UPI" or "Dispatch Police" actions are simulated.
- Deep Neural Net verification is out-of-scope for the Counterfeit scanner to ensure offline deterministic execution.
