# Demo Checklist

- [ ] Repository cloned and on the `main` branch.
- [ ] Dependencies installed (`make setup`).
- [ ] Database is clean (`./verify.ps1` runs seed successfully).
- [ ] Backend is running on port 8001 (`make dev`).
- [ ] Frontend is running on port 3000 (`make dev`).
- [ ] Screen resolution is set appropriately for MapLibre/Cytoscape visualizations.
- [ ] Pre-load `http://localhost:3000/demo-control` in one tab.
- [ ] Pre-load `http://localhost:3000` (Dashboard) in another tab.
- [ ] Have a sample low-quality image ready for Counterfeit scanner drag-and-drop.
